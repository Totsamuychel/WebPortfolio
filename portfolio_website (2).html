CopyTool App with AI

Интеллектуальный инструмент для автоматического распознавания и копирования текста с экрана

Python Computer Vision OCR Local AI

🎯 Проблема и решение

❌ Проблема

Необходимость быстро копировать текст с экрана без ручного выделения. Существующие OCR решения требуют загрузки в облако или работают медленно.

✅ Решение

Локальная AI модель мгновенно распознает текст с экрана и копирует в буфер обмена. Полная приватность и скорость <200ms.

🚀 Быстрый старт

  1. Скачайте релиз с GitHub
  2. Установите зависимости: pip install -r requirements.txt
  3. Запустите: python copytool.py
  4. Выделите текст на экране и нажмите горячую клавишу
  5. Текст автоматически скопируется в буфер обмена!

# Инициализация CopyTool с AI моделью from copytool import AITextDetector # Создание детектора detector = AITextDetector(model='local_ocr_model') # Запуск распознавания detected_text = detector.scan_screen() detector.copy_to_clipboard(detected_text) print(f"Скопировано: {detected_text}")
