Google Account Sorter — Automated Sheet Management

Автоматическая сортировка данных в Google Sheets с поддержкой множества аккаунтов.

Python View on GitHub →

About the Project

Инструмент для автоматизации работы с большими объемами данных в облачных таблицах. Позволяет управлять сотнями таблиц одновременно.

How It Works

Использует Google Sheets API и OAuth2 для безопасного доступа к данным. Скрипт анализирует структуру входных данных и через gspread выполняет пакетные обновления таблиц.

Technologies Used

Python

Key Features

  • ✔ Поддержка сотен Google-аккаунтов
  • ✔ Пакетная обработка данных
  • ✔ Безопасная OAuth2 авторизация
  • ✔ Гибкая настройка правил сортировки

Code Implementation

import gspread

def sort_data(sheet_id):
    gc = gspread.service_account()
    sh = gc.open_by_key(sheet_id)
    # Логика сортировки данных
    sh.worksheet('Main').sort((1, 'asc'))
