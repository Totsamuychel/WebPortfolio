CopyTool — AI Screen Text Detector

Использование локальной ИИ-модели для детекции текста на экране и копирования.

Python View on GitHub →

About the Project

Утилита для мгновенного захвата текста из любой области экрана. Работает на базе локальной OCR и LLM для улучшения качества распознавания.

How It Works

При нажатии горячей клавиши программа делает скриншот выделенной области (Pillow + PyAutoGUI), отправляет его в локальную модель для распознавания текста и автоматически очищает результат от мусора перед помещением в буфер обмена.

Technologies Used

Python OpenCV

Key Features

  • ✔ Мгновенный захват области экрана
  • ✔ Локальное распознавание текста (OCR)
  • ✔ Интеллектуальная очистка через LLM
  • ✔ Минимальное потребление ресурсов
