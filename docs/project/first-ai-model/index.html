About the Project

Базовый, но важный проект, с которого начался мой путь в Deep Learning. Это классическая реализация нейронной сети для распознавания цифр (MNIST), позволяющая понять основы обучения моделей, градиентного спуска и работы с тензорами.

How It Works

Используется библиотека TensorFlow или PyTorch для построения архитектуры CNN (Convolutional Neural Network). Модель обучается на наборе данных MNIST и достигает высокой точности распознавания уже после нескольких эпох обучения.
